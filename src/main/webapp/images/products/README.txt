Product Images Directory Structure
==============================

This directory contains product images organized by category:

/men - Men's clothing and accessories
/women - Women's clothing and accessories
/kids - Kids' clothing and accessories
/accessories - General accessories

Image Naming Convention
----------------------
Please name your product images using the following format:
[category]_[product-type]_[number].jpg

Examples:
- men_tshirt_1.jpg
- women_dress_2.jpg
- kids_jeans_1.jpg
- accessories_belt_1.jpg

Image Requirements
-----------------
- Format: JPG or PNG
- Size: 800x800 pixels (square) recommended
- Max file size: 500KB

Placeholder Images
-----------------
For placeholder images, you can use:
- https://placehold.co/800x800?text=Product+Name
- https://picsum.photos/800/800

Example: You can download a placeholder image from https://picsum.photos/800/800
and save it as placeholder.jpg in this directory.
