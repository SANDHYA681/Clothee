Product Images Setup

Quick Start

To set up product images for the Clothee e-commerce application, you have two options:

Download Real Images Generate Placeholder Images

Setup Instructions

Option 1: Download Real Images

  1. Open download_images.html in your web browser
  2. Click the "Download" button for each image
  3. Save the image with the specified filename (e.g., tshirt_1.jpg)
  4. Place the image in the appropriate category folder

Option 2: Generate Placeholder Images

  1. Open generate_placeholders.html in your web browser
  2. Click the "Generate" button for each category
  3. Right-click on each generated image and select "Save Image As..."
  4. Save with the specified filename (e.g., tshirt_1.jpg)
  5. Place in the appropriate category folder

Database Setup

After placing the images in the correct directories, you need to update the database:

  1. Run the product_images.sql script to create the product_images table
  2. Run the sample_product_images.sql script to add the image references to the database
-- Run these commands in your MySQL database
source product_images.sql
source sample_product_images.sql

Verifying the Setup

To verify that the images are correctly set up:

  1. Start the Tomcat server
  2. Navigate to the products page
  3. Click on a product to view its details
  4. Verify that the images are displayed correctly
  5. Test the image gallery functionality by clicking on thumbnails

If you encounter any issues with the images, check the following:

  • Ensure the image files are in the correct directories
  • Verify that the file names match those in the database
  • Check the browser console for any errors related to image loading
  • Ensure the image paths in the database are correct
